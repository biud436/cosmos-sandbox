# Cosmos

> 웹 기반 인터랙티브 입자 샌드박스 — 브라우저에서 분자 동역학, 화학, 자기중력, 미니 블랙홀, 간이 핵융합을 시뮬레이션합니다.

[English README](./README.md)

Cosmos는 고전 물리 아래에서 입자 집단의 시간 변화를 시뮬레이션합니다: Lennard-Jones 분자간 힘, 조화 화학 결합, 뉴턴식 자기중력, 임계값 기반 단순화 핵융합. 시간은 일시정지·단계 진행·배속이 가능하며, 활성 프리셋에 따라 만/억/조 년 단위의 우주 시간으로 표시됩니다.

뷰포트는 Unity 스타일 5분할 레이아웃입니다: 툴바 · Hierarchy · Viewport · Inspector · Console.

## 기능

### 물리
- **Lennard-Jones 쌍 힘** — 종 간 Lorentz–Berthelot 혼합 규칙 사용.
- **Velocity Verlet 적분** + 속도 재조정(velocity-rescaling) 서모스탯.
- **자기중력** (softened Newtonian, O(N²)) — 우주 스케일 응집.
- **화학 결합**: 같은 종 원자 간 조화 스프링, 가전자 한도 제약. 가까이 접근하면 형성, 과도하게 늘어나면 파괴.
- **간이 핵융합**: H + H → He, 상대 운동에너지가 임계값을 넘으면 발생 (운동량 보존 + 에너지 방출).
- **블랙홀**: 드래그로 배치하는 중력 우물. event horizon 내부 입자는 흡수됨.
- **경계**: 반사형 박스(엣지 와이어프레임으로 시각화) 또는 열린 공간 — 프리셋에 따라 다름.

### 입자 종
| 종 | 질량 | σ | ε | 가전자 | 비고 |
|---|---|---|---|---|---|
| H    | 0.05 | 0.55 | 0.35 | 1 | H₂ 형성, He로 융합 가능 |
| He   | 0.10 | 0.65 | 0.20 | 0 | 비활성 기체 |
| N₂   | 0.70 | 1.00 | 1.00 | 0 | 이미 분자 |
| O₂   | 0.80 | 1.00 | 1.10 | 0 | 이미 분자 |
| Dust | 5.00 | 1.30 | 2.20 | 4 | 클러스터 형성 가능 |

매 프레임 결합 그래프에 Union-Find를 적용해 실제 존재하는 분자(예: `H₂`, `Dust ×4`)를 Hierarchy 패널에 표시합니다.

### 시간 모델
- **Time scale**: ×0.25 / ×0.5 / ×1 / ×2 / ×4 / ×8 (프레임당 서브스텝 수). 내부 `dt`는 고정.
- **Pause** (Space) — 시뮬레이션 시간을 정지. **Step** (`.`) — 1틱 전진.
- **우주 시간 표시**: 프리셋별 `yearsPerUnit`이 환산 시간을 년 단위로 매핑하고 만/억/조 단위로 한국어 포맷팅.

### 프리셋

씬 단위 프리셋:
- **우주 가스 구름** — 자기중력·결합 활성 상태의 차가운 H + He. H₂가 형성되고 가스 구름이 수백만 년에 걸쳐 응집.
- **공기 흐름** — 바람이 부는 N₂/O₂ + 먼지, 결합 활성. 먼지가 사슬 형태로 응집.
- **저온 응축** — 결합 활성 + 저온 가스. 응축 액적 형성.
- **항성 내부 (융합)** — 고온·고밀도 H. 결합은 즉시 깨지고 핵융합이 우세.

각 Inspector 폴더(환경 / 입자 구성 / 화학 결합 / 핵융합)에는 부분 적용을 위한 자체 서브 프리셋 드롭다운이 있습니다.

### 도구
- **Black Hole** 버튼 — Tools 패널에서 뷰포트로 드래그. 버튼 자체가 회전하는 강착원반 애니메이션을 끊임없이 그리고, 같은 모양의 고스트가 커서를 따라다님. **Esc**로 드래그 취소. **Clear**로 모든 블랙홀 제거.

## 컨트롤

| 동작 | 키 / UI |
|---|---|
| Pause / Resume | 툴바 `⏸ Pause` · `Space` |
| 단발 전진 | `⏭ Step` (일시정지 시) · `.` |
| Reset (현재 분포로 재생성) | `↺ Reset` |
| 프리셋 / 시간 배속 | 툴바 드롭다운 |
| 블랙홀 드래그 취소 | `Esc` |
| 카메라 | 마우스 드래그 · 스크롤 휠 (OrbitControls) |

## 아키텍처

```
src/
├── main.ts                전체 연결 · 프레임 루프
├── physics/
│   ├── types.ts           종 테이블 (mass/σ/ε/color/valence)
│   ├── SpatialGrid.ts     O(N) 셀-리스트 이웃 탐색
│   └── Simulator.ts       LJ + Verlet + 서모스탯 + 결합 + 융합 + 블랙홀
├── render/
│   └── Scene.ts           Three.js 씬: 별/네뷸라 배경, InstancedMesh / Points 입자,
│                          결합 LineSegments, 블랙홀 셰이더, 화면→월드 좌표 헬퍼
└── ui/
    ├── presets.ts         상위 씬 프리셋
    ├── subPresets.ts      Inspector 폴더별 서브 프리셋
    ├── Controls.ts        lil-gui Inspector
    ├── Layout.ts          Hierarchy / Console / 툴바 바인딩
    ├── Tools.ts           블랙홀 도구: 캔버스 애니메이션 + 드래그 핸들러
    └── timeFormat.ts      우주 시간 포매터
```

물리는 내부적으로 환산(LJ) 단위. 온도는 보정 상수를 통해 K 단위로 노출. 우주 시간은 프리셋별 `yearsPerUnit` 배수로 환산.
